# vulkan-cpp-demo

#### 介绍
vulkan学习和c++封装库

#### 软件架构
软件架构说明
一个基于vulkan的渲染框架


#### 安装教程

1.  安装vulkan sdk
2.  安装cmake
3.  安装gcc 后即可编译
根目录下执行 cmake . + make即可
